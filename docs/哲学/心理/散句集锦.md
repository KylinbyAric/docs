# 人不是因为自卑而不敢行动，而是为了不行动，才选择了自卑。

## 杀死心中的神，允许自己的普通。
这可能会被误会为不上进，但人只有在觉得自己普通的时候，才会是白纸状态，才能接纳更多。
允许自己搞砸，允许自己被拒绝，允许自己被讨厌，允许自己碌碌无为。破除完美心态。
PS:就跟玩法环，如果我追求所有支线完美执行，那这个游戏将变得痛苦无比，最终导致玩不下去。

## 内部记分牌
即不管外部评价如何，自己要接受和欣赏自己的小进步，外部眼光被养得太挑剔了，是看不到自己的小进步的。
社会微不足道甚至毫无价值的小进步，对于个人而言却是踏上月球的第一步。

## 羞耻脱敏
刻意做一些自己觉得羞耻或者轻微尴尬的事，看看外界的反应。这就是所谓自嘲？自我调侃也是一种自信。
年会上，will在开幕说起孙枢委屈地都哭了。孙枢之后第一句话就是自我调侃说这次也要哭一下，看看业绩能不能像上次那样再涨涨。
没实践过，可以实践一下。以前倒是很喜欢自黑的，后面就少了。
