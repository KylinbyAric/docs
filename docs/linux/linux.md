多级反馈队列：
1、设置多个就绪队列，优先权越高的时间片越短。
2、新进程进入第一队，先进先出服务，如果在时间片内完成就退出，没完成就进入第二队列，然后同样操作。知道第n队列。
3、只有处理完前一个队列才会继续处理下一个队列。
4、如果有优先级更高的队列进入了线程，则新队列抢占CPU，被中断的进程进入之前队列的对尾。
